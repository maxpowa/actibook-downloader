# Actibook Downloader

This project generates a userscript which will assist in download/archive of magazines/comics accessed through an actibooks UI.

### Click [here](https://github.com/maxpowa/actibook-downloader/raw/main/dist/actibookdl.user.js) to install

If the script is working, you will see a new green download icon next to the bottom controls button, near the bottom right of the screen.
